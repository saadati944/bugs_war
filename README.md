# Bugs War

control your bug with writing codes
every bug has 6 operations

- `bug.move_right()`
- `bug.move_left()`
- `bug.move_up()`
- `bug.move_down()`
- `bug.scan(deg)` # returns a boolean
- `bug.shoot(deg)`

if you want to check your health, then use this :
- `bug.health`

the position of the bug:
- `bug.X`
- `bug.Y`

world size:
- `bug.width`
- `bug.height`

# how to run ?

```bash
python bugswar.py
```
